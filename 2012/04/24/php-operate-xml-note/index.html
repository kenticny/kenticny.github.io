对于 XML 的操作包括遍历，生成，修改，删除等其他类似的操作。PHP 对于 XML 的操作方式很多，这次学习的是通过 DOMDocument 进行操作，其他的操作方法可以参考http://www.oschina.net/code/snippet_110138_4727
